# Criar novo usuario

## Caso de Sucesso

- Sistem executa o comado 'registrar usuario'
- Sistema valida os dados de entrada
- Sistema verifica se o usuario ja esta cadastrado
- Sistema cria uma wallet padrão

## Exceção - Usuario ja esta cadastrado

- Sistema retorna um erro

## Exceção - Erro de validação

- Sistema retorna erro de validação contendo os campos errados
